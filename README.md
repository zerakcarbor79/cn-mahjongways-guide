# cn-mahjongways-guide

本仓库用于归档和发布多个独立的 HTML 页面。所有页面均为静态内容，不依赖后端服务，可直接在浏览器中打开查看。

## 项目简介

本项目旨在整理与麻将玩法、规则、文化相关的信息，并将其以独立 HTML 页面的形式集中存放。每个页面都是自包含的，便于分发、离线查看或嵌入到其他内容中。

## 目录说明

仓库根目录下存放各 HTML 文件，文件名以英文或拼音命名，简要反映页面主题。例如：

- `index.html` — 首页或导航页
- `rules-basic.html` — 基础规则说明
- `scoring.html` — 计分方式介绍

实际文件列表以仓库中最新提交为准。每个页面均包含完整的样式与内容，无需额外资源文件。

## 页面归档说明

- 所有页面均为静态 HTML，无外部依赖。
- 页面内容可能随时间更新，以仓库主分支最新版本为准。
- 部分页面可能包含示例性数据或占位内容，仅供学习参考。

## 维护说明

- 欢迎通过 Issue 提交内容建议或报告问题。
- 不接受直接提交包含商业推广、外链或诱导点击内容的 Pull Request。
- 维护者会定期检查并更新页面内容，确保信息准确。
- 本仓库不保证页面内容适用于所有地区或规则体系。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开任意 `.html` 文件即可查看。
3. 也可通过 GitHub Pages 等服务部署后在线访问（需自行配置）。
